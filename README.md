# 🍗 Hứng Khô Gà -

Một trò chơi giải trí nhẹ nhàng được xây dựng bằng **Kaplay.js** (Kaboom.js v3000+), lấy cảm hứng từ "Tộc trưởng" Độ Mixi và những miếng Khô gà thần thánh.

##  Tính năng nổi bật
- **Survival Mode (5 Phút)**: Thử thách sinh tồn trong 5 phút để nhận phần thưởng "1 tấn Khô gà".
- **Độ khó tăng dần**: Càng về sau các vật phẩm rơi càng nhanh, đòi hỏi phản xạ cực tốt.
- **Hệ thống vật phẩm**:
  -  **Khô Gà**: +10 điểm.
  -  **Dép Lào**: Trừ 1 mạng (Có 3 mạng khởi đầu).
  -  **Bom**: Game Over ngay lập tức.
- **Âm thanh & Hiệu ứng**: 
  - Nhạc nền sôi động (F1 đè tem).
  - Hiệu ứng rung màn hình, âm thanh va chạm sinh động.
- **Tiện ích**: Hỗ trợ tạm dừng game (Pause - Phím Space/P) và tắt nhạc nền nhanh.
- **Kỷ lục**: Lưu điểm cao nhất vào Local Storage.

## 🛠 Công nghệ sử dụng
- **Ngôn ngữ**: Javascript (ES6+)
- **Engine**: [Kaplay.js](https://kaplayjs.com/)
- **Styling**: Bản vẽ Canvas nội trú & CSS3 cơ bản.

##  Cách chạy Game
1. Clone repository này về máy.
2. Mở file `index.html` trực tiếp trên trình duyệt hoặc sử dụng **Live Server*

*.
3. Sử dụng **Chuột** hoặc **Phím Mũi tên** để di chuyển rổ (Cairo) hứng đồ.

##  Cấu trúc thư mục
- `asset/`: Chứa toàn bộ hình ảnh và âm thanh của trò chơi.
- `js/game.js`: File logic chính quản lý toàn bộ trò chơi.
- `index.html`: Điểm khởi đầu của ứng dụng.
